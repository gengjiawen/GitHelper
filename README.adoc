== What can it do
download github or other git repository to a 7z file with _.git_ folder

== Dependencies
Basically, you need to install https://git-scm.com/[git] and http://www.7-zip.org/[7z]

For ubuntu
[source]
sudo apt-get install git p7zip-full

== How to play
[source, python]
python3 MyFlask.py
